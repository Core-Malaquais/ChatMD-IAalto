Pour vous connecter à [Taïga](https://etudiant.archi.fr/taiga/etd/index.php?ce=malq), utilisez `prénom.nom` comme identifiant et saisissez votre mot de passe qui est le même que sur Office 365.

\`
:::info
Si vous n'avez pas de mot de passe, lancez la procédure de mot de passe oublié ou contactez le service pédagogique.
:::
\`

## taiga_document
- taïga document
- certificat de scolarité

Pour télécharger votre document administratif dans [Taïga](https://etudiant.archi.fr/taiga/etd/index.php?ce=malq), rendez-vous dans le menu `Global -> Données personnelles`.

1. [Comment se connecter à Taïga ?](taiga_connexion)

## taiga_note
- taïga note
- relevé de note
- bulletin de note

Pour visualiser votre relevé de note dans [Taïga](https://etudiant.archi.fr/taiga/etd/index.php?ce=malq), rendez-vous dans le menu `Global -> Relevé de note`.

\`
:::info
Vos notes ne sont visibles qu'une fois saisies par votre enseignant référent.
:::
\`

1. [Comment se connecter à Taïga ?](taiga_connexion)
