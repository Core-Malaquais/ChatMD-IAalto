# ChatMD-IAalto

[https://chatmd.forge.apps.education.fr///#https://raw.githubusercontent.com/Core-Malaquais/ChatMD-IAalto/refs/heads/main/DialogueMD.md](IAalto) est un chatbot à destination des usagers de l'ENSA Paris-Malaquais basé sur l'outil [https://chatmd.forge.apps.education.fr](ChatMD) créé par Cédric Eyssette, professeur de philosophie et chargé de projet à la DRANE à Lyon.
